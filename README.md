# 수학 퀴즈 - 중학교 수학학원용 퀴즈 시스템

학생들이 iPad에서 수학 문제를 풀고, 결과가 자동으로 선생님 Google Sheets에 저장되는 무료 웹앱입니다.

## 주요 기능

- 중1/중2/중3 전 학년 **18개 단원** 문제 자동 생성
- **객관식 + 주관식** 모두 지원
- **수식 렌더링** (KaTeX) + **도형 그림** (SVG) 자동 생성
- 관리자 페이지에서 **원클릭 퀴즈 생성**
- 학생 결과 **Google Sheets 자동 저장**
- 비용: **완전 무료** (GitHub Pages + Google Apps Script)

## 지원 단원

| 중1 | 중2 | 중3 |
|-----|-----|-----|
| 소인수분해 | 식의 계산 | 제곱근과 실수 |
| 정수와 유리수 | 연립방정식 | 인수분해 |
| 일차방정식 | 부등식 | 이차방정식 |
| 좌표와 그래프 | 일차함수 | 이차함수 |
| 기본 도형 | 도형의 성질 | 삼각비 |
| | 확률 | 원의 성질 |
| | | 통계 |

## 설치 가이드 (비개발자용)

### 1단계: 내 컴퓨터에 프로젝트 받기

1. https://github.com/dongmin1213/math-quizzes 에 접속
2. 초록색 **Code** 버튼 클릭 → **Download ZIP** 클릭
3. 다운로드된 ZIP 파일 압축 풀기

또는 터미널에서:
```bash
git clone https://github.com/dongmin1213/math-quizzes.git
```

### 2단계: 로컬에서 테스트하기 (인터넷 없이도 가능)

1. 터미널(맥: Terminal, 윈도우: PowerShell)을 열고:
```bash
cd 압축푼폴더/math-quizzes
python3 -m http.server 8080
```
2. 브라우저에서 http://localhost:8080/admin.html 접속
3. 비밀번호: `math1234`
4. 학년/단원 선택 → **문제 생성하기** → 미리보기 확인 → **배포하기**
5. http://localhost:8080/ 에서 학생 화면 테스트

### 3단계: GitHub Pages로 인터넷에 공개하기

1. GitHub 계정 생성 (https://github.com)
2. 새 리포지토리 생성 (예: `math-quiz`)
3. 파일을 업로드하거나 git push
4. 리포지토리 **Settings** → **Pages** → Source를 **Deploy from a branch** / **main** 선택
5. 몇 분 후 `https://내아이디.github.io/math-quiz/` 에서 접속 가능

### 4단계: Google Sheets 연동하기 (선택사항)

> Google Sheets를 연동하면 학생 결과가 자동 저장됩니다.
> 연동하지 않아도 퀴즈 생성/풀기는 정상 동작합니다.

1. Google Drive에서 **새 스프레드시트** 생성 (이름: "수학퀴즈 결과")
2. 상단 메뉴: **확장 프로그램** → **Apps Script** 클릭
3. 기본 코드를 모두 지우고, 이 프로젝트의 `apps-script/Code.gs` 내용을 **전체 복사**하여 붙여넣기
4. **배포** → **새 배포** 클릭
5. 유형: **웹 앱** 선택
6. **실행 주체**: "나" / **액세스 권한**: "누구나" 선택
7. **배포** 클릭 → URL 복사 (예: `https://script.google.com/macros/s/xxxxx/exec`)
8. 프로젝트의 `js/config.js` 파일을 열고:
```js
APPS_SCRIPT_URL: '여기에 복사한 URL 붙여넣기',
```
9. 저장 후 GitHub에 다시 업로드

### 비밀번호 변경

`js/config.js` 파일에서:
```js
ADMIN_PASSWORD: 'math1234',  // ← 원하는 비밀번호로 변경
```

## 사용 방법

### 선생님 (관리자)

1. `admin.html` 접속 → 비밀번호 입력
2. **퀴즈 생성** 탭:
   - 학년 선택 → 단원 선택 → 문제 수 / 난이도 / 객관식 비율 설정
   - **문제 생성하기** 클릭 → 미리보기에서 문제 확인
   - **배포하기** 클릭 → 학생들이 풀 수 있는 상태가 됨
3. **결과 보기** 탭 (Google Sheets 연동 시):
   - 학생별 점수, 정답률, 오답 문제 확인

### 학생

1. 메인 페이지 접속 → 이름 / 학년 / 반 입력
2. **퀴즈 시작하기** 클릭
3. 문제 풀기 (상단 점으로 이동 가능, 이전/다음 버튼)
4. **제출하기** 클릭 → 결과 확인 (정답/오답/해설)

## AI로 기능 추가하는 방법

이 프로젝트는 AI 도구(Claude, Gemini 등)로 쉽게 확장할 수 있도록 설계되었습니다.

### Claude Code 사용 시

프로젝트 폴더에서 `claude` 명령어를 실행하면 `CLAUDE.md` 파일을 자동으로 읽어서 프로젝트 구조와 코딩 규칙을 이해합니다.

```bash
cd math-quizzes
claude
```

### 새 단원 추가를 요청하는 방법 (예시)

> "중1 통계 단원 생성기를 추가해줘. 기존 생성기 패턴(IIFE, 역생성, 난이도 3단계)을 따라서 만들어줘. config.js와 admin.html에도 등록해줘."

### 디자인 변경 (예시)

> "결과 페이지에서 오답 문제만 모아서 다시 풀기 기능을 추가해줘."

> "관리자 페이지에 학생별 성적 추이 그래프를 추가해줘."

### 기능 확장 (예시)

> "타이머 기능을 추가해줘. 문제당 2분 제한시간이 있고, 시간이 지나면 자동으로 다음 문제로 넘어가게 해줘."

> "학생이 로그인 없이 이전에 푼 퀴즈 기록을 볼 수 있게 해줘."

### AI에게 요청할 때 팁

1. **"기존 패턴을 따라서"** 라고 명시하면 일관된 코드를 생성합니다
2. **구체적인 요구사항**을 적으면 더 정확한 결과를 얻습니다
3. 작업 후 **"로컬에서 테스트해줘"** 라고 하면 문제를 미리 발견할 수 있습니다
4. `CLAUDE.md`가 있으면 AI가 알아서 코딩 규칙을 지킵니다
